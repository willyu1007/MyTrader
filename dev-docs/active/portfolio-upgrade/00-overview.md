# 00 Overview

## Status
- State: in_progress
- Next step: 推进数据分析多场景（优先：指数/板块），补齐“数据分析”入口下的非组合分析能力

## Goal
完善“投资组合”模块，覆盖：资产与账户、交易流水、持仓引擎、估值与收益率、归因与风险、再平衡、报表，并与数据管道（行情/公司行为/汇率/日历）形成闭环。

## Non-goals
- 自动下单/券商直连/交易执行
- 云端同步、多设备共享、多用户协作
- 在仓库内保存任何 Token/密钥

## Acceptance criteria（高层）
- [ ] 顶层 Tabs 与信息架构稳定，用户可按任务路径完成“导入/拉取 -> 计算 -> 分析 -> 报告”
- [ ] 交易流水/现金流可用，并可驱动（或至少对齐）持仓引擎的结果
- [ ] 估值/收益率/风险等指标口径明确，且每个结果可追溯到输入数据与时间（as-of/version）
- [ ] 数据缺失/过期/异常可解释，并提供修复路径（一键刷新/导入）
